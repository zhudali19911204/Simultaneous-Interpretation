Teams 中英同声翻译助手 - Windows x64
====================================

推荐安装方式
------------
1. 先完整解压 ZIP 文件，不能直接在压缩包预览中运行。
2. 双击 Install.cmd。
3. 程序会安装到当前用户的：
   %LOCALAPPDATA%\TeamsInterpreter
4. 安装程序会创建桌面和开始菜单快捷方式，不需要管理员权限。

免安装运行
----------
直接打开 app 文件夹，双击 TeamsInterpreter.exe。

首次使用
--------
1. 点击右上角“设置”，填写同传 API Key、WorkspaceId 和会议纪要模型配置。
2. API Key 保存在当前 Windows 用户的凭据管理器中，不包含在安装包内。
3. 在“音频路由”中选择实体麦克风和 Teams 扬声器回放捕获。“英文语音输出”默认关闭，同时列出检测到的实体设备和虚拟声卡；需要合成英文声音时，再选择目标设备。

PPT 演示字幕
------------
1. 开始同传后，点击“显示演示字幕”，或按 Ctrl+Shift+S。
2. 中文固定显示在上方，英文固定显示在下方。
3. 字幕保留最近 3 组最终结果并自动滚动；未完成句超时只会清除当前行，不会删除已完成译文。
4. 按住字幕条可拖动位置，双击可隐藏。
5. 需要让 Teams 参会者看到字幕时，请共享整个屏幕；只共享 PowerPoint 窗口通常不会包含字幕层。

AI 会议理解助手
----------------
1. 点击主窗口底部“会议助手”，可查看按时间排列的完整会议字幕并搜索、筛选和复制；查看时间线不调用网络。
2. “会中问答”和“实时重点”复用设置中的会议纪要模型，调用会产生相应 Token 费用。
3. 自动重点默认关闭；主动开启后，每 5 分钟且有至少 300 个新字符时才更新。
4. 助手错误不会停止同传。关闭助手会停止自动刷新，“清空记录”会同步清空助手内容。
5. 助手内容仅保存在当前应用内存中，不会自动写入文件。

AI 会议纪要与关键截图
--------------------
1. 停止同传后点击“生成 AI 会议纪要”。
2. 存在共享画面时，纪要窗口会提示可导出的关键截图数量；点击“保存纪要和截图”。
3. 应用会保存一个 Markdown 文件和同名 _assets 图片目录，最多自动筛选 20 张关键截图。
4. 分享或移动纪要时，必须同时携带 Markdown 和对应的 _assets 目录，否则图片链接会失效。
5. 同名图片目录已存在时会自动使用 _assets_2、_assets_3 等新目录，不覆盖原附件。
6. 没有共享画面时仍只保存原来的单个 Markdown 文件；“复制文字”不会复制本地图片链接。

AI 共享画面理解
---------------
1. 在“设置”→“共享画面 AI”中配置支持 OpenAI image_url 的视觉模型；百炼默认模型为 qwen3-vl-plus。
2. 开始同传后，打开“会议助手”→“共享画面”，点击“选择区域并开始”，拖选 Teams 或 PPT 共享区域。
3. 页面稳定约 2 秒后会自动生成中文摘要、关键数字、术语、行动项、风险和未决问题。
4. 自动分析最短间隔 15 秒，每小时最多 60 次；繁忙时只保留最新页面。可随时手动“分析当前页”。
5. Ctrl+Alt+F9 可全局暂停/恢复；热键不可用时使用窗口按钮。切换共享来源时点击“重新选择”。
6. 中文速读浮窗默认关闭；开启后采样时会临时隐藏，避免捕获自身。
7. 截图默认只保存在内存并发送给所选视觉服务，不创建临时文件。最多保留 20 页完整压缩图并优先保留纪要关键页，最近 120 页保留缩略图；只有主动保存纪要时才会把关键截图写入指定目录。
8. 停止同传或关闭会议助手会停止捕获；“清空记录”会同时清除视觉记录和内存图片。

断线恢复
--------
1. 首次启动遇到短暂 DNS、超时、429 或服务端错误时，会在启动窗口内自动重试。
2. 运行中单个翻译方向断开后会持续后台重连，另一个方向保持工作。
3. 主窗口和演示字幕会显示重连方向与次数，恢复后提示自动消失。
4. 已完成字幕和会议记录会保留；断线期间的音频不会补传。
5. 持续 DNS 错误会提示检查北京 WorkspaceId、VPN、代理、公司 DNS 或手机热点。
6. 诊断日志位于 %APPDATA%\SimultaneousInterpreter\logs\interpreter.log，不包含凭据或会议内容。

重要说明
--------
- 应用本身不需要 Python，也不需要管理员权限。
- 若要把合成英文语音作为 Teams 麦克风发送，仍需另外安装 VB-CABLE 或兼容虚拟声卡。
- 虚拟声卡驱动的安装通常需要管理员权限，安装包不包含该第三方驱动。
- Windows SmartScreen 可能提示“未知发布者”，因为此内部版本没有购买代码签名证书。
- 请通过可信的内部渠道分发，并用 BUILD-INFO.txt 或发送者提供的 SHA256 校验文件。

卸载
----
1. 退出 Teams Interpreter。
2. 删除 %LOCALAPPDATA%\TeamsInterpreter 文件夹。
3. 删除桌面和开始菜单中的 Teams Interpreter 快捷方式。
4. 如需同时删除凭据，可先在应用“设置”中点击“清除凭据”。
